PhishProof QR Demo Site

What this is
A lightweight front-end demo for the PhishProof QR concept. It is designed to help with hackathon pitching and walkthroughs before the full iOS app exists.

What it demonstrates
1. The product story
2. URL risk analysis
3. Redirect-chain explanation
4. Auto-continue behavior for safe links

Files
- index.html
- styles.css
- app.js

How to use it
Open index.html in a browser.

Demo tips
1. Start with the safe sample to show the low-risk flow.
2. Turn on auto-continue safe actions.
3. Show that safe links can continue without rescanning.
4. Then switch to the phishing samples to show the warning logic.
