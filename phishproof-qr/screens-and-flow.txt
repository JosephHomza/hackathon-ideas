Screens And Flow

Screen 1: Home
- App title
- short explanation
- scan now button
- import image button

Screen 2: Scanner
- live camera preview
- corner guide overlay
- flashlight toggle
- cancel button

Screen 3: Analysis
- loading animation
- messages like:
  - decoding QR
  - checking redirects
  - analyzing domain

Screen 4: Results
- verdict badge
- payload type
- final URL
- redirect chain
- risk reasons
- recommendation text
- continue safely button
- copy link button
- save result button

Screen 4A: Action Preview
- decoded action summary
- final URL or payload details
- warning if risky
- confirm action button
- cancel button

Screen 5: History
- scan list
- verdict color
- timestamp
- tap to reopen result

Screen 6: Settings
- privacy notice
- clear history
- about
- optional API settings
- auto-execute safe actions toggle
- allowed auto-action types

Core flow
Home -> Scanner -> Analysis -> Results -> Action Preview -> Continue or Save

Secondary flow
Home -> Import Image -> Analysis -> Results

Optional fast flow
Home -> Scanner -> Analysis -> Auto-continue if Safe and enabled
